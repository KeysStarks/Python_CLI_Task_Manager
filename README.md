# Python CLI Task Manager

## Overview

Python CLI Task Manager is a command-line application that allows users to manage tasks directly from the terminal.

The application was built using Python and demonstrates core programming concepts including:

- Variables
- Lists
- Dictionaries
- Functions
- Loops
- Conditional Statements
- Exception Handling
- JSON File Storage
- Data Persistence

This project was originally created as a Coding Temple assignment and was expanded with additional functionality beyond the assignment requirements.

---

## Features

### Task Management

- Add Tasks
- View Tasks
- Delete Tasks
- Mark Tasks Complete
- View Completed Tasks

### Validation & Error Handling

- Prevents empty tasks
- Handles invalid menu selections
- Handles invalid task numbers
- Handles attempts to delete non-existent tasks

### Data Persistence

Tasks are automatically saved to a JSON file and loaded when the application starts, allowing users to retain their tasks between sessions.

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate into the project directory

```bash
cd python-cli-task-manager
```

3. Run the application

```bash
python3 todo_app.py
```
